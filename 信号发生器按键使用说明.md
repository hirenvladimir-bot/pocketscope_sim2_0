# PocketScope 信号发生器 — 按键使用说明

## 概述

信号发生器是 PocketScope 的四大功能模式之一（模式选择 `sw[1:0] = 00`），基于 DDS（直接数字频率合成）技术，通过 DAC0832 输出双极性模拟信号。支持正弦波、方波、三角波三种基本波形，以及 AM 调幅、FM 调频、SPWM 正弦脉宽调制三种调制模式。输出幅度以有效值（Vrms）标定，0–2Vrms 可调。

---

## 一、按键与开关布局

EGO1 开发板上的控制资源分配如下：

| 硬件资源 | 功能 | 说明 |
|:---:|:---|:---|
| **sw[1:0]** | 主模式选择 | 00=信号发生器 |
| **sw[4:2]** | 子模式 / 波形类型 | 见下方波形选择表 |
| **sw[7:5]** | 频率粗调档位 | 5 个档位，见下方频率设置表 |
| **DIP[7:0]** | 频率微调 | 8 位拨码开关，0~255 级 |
| **PB0 (btn[0])** | 幅度 + | 每按一次幅度 +10（约 +78mVrms） |
| **PB1 (btn[1])** | 幅度 − | 每按一次幅度 −10（约 −78mVrms） |
| **PB2 (btn[2])** | 调制深度 + | 每按一次 +16 |
| **PB3 (btn[3])** | 调制深度 − | 每按一次 −16 |
| **PB4 (btn[4])** | （示波器触发调节） | 信号发生器模式下不使用 |

---

## 二、进入信号发生器模式

将 **sw[1:0]** 拨到 **00**，即可进入信号发生器模式。

- VGA 屏幕显示：深色背景 + 灰色网格 + 中心十字线
- 绿色波形轨迹实时预览
- LED[0] 点亮，表示当前处于信号发生器模式

---

## 三、波形选择

通过 **sw[4:2]** 三位拨码开关选择输出波形：

| sw[4:2] | 波形 / 模式 | 说明 |
|:---:|:---|:---|
| **000** | 正弦波 (Sine) | 默认波形，从 sine_lut.hex 查表输出，中心电平 0x80 |
| **001** | 方波 (Square) | 对称方波，满幅摆动（±127），占空比 50% |
| **010** | 三角波 (Triangle) | 对称三角波，中心电平 0x80，线性上升/下降 |
| **011** | **AM 调幅** | 用调制波（正弦）调制载波幅度 |
| **100** | **FM 调频** | 用调制波（正弦）调制载波频率 |
| **101** | **SPWM 正弦脉宽调制** | 载波与三角波比较，输出 PWM 信号 |

> **注意**：sw[4:2] 设为 011/100/101 时自动启用调制（mod_enable=1），其余波形调制关闭。

---

## 四、频率设置

信号频率由 **粗调档位（sw[7:5]）** + **微调拨码（DIP[7:0]）** 共同决定。

### 频率计算公式

```
输出频率 (Hz) = 档位基频 + DIP值 × 20 Hz
```

### 粗调档位表

| sw[7:5] | 基频 | DIP 调节范围 | 总频率范围 |
|:---:|:---:|:---|:---|
| **000** | 100 Hz | 0 ~ 5100 Hz | 100 Hz ~ 5.2 kHz |
| **001** | 2000 Hz | 0 ~ 5100 Hz | 2 kHz ~ 7.1 kHz |
| **010** | 4000 Hz | 0 ~ 5100 Hz | 4 kHz ~ 9.1 kHz |
| **011** | 6000 Hz | 0 ~ 5100 Hz | 6 kHz ~ 11.1 kHz |
| **100** | 8000 Hz | 0 ~ 5100 Hz | 8 kHz ~ 13.1 kHz |

### 使用示例

- 想输出 **1 kHz 正弦波**：sw[7:5] = 000，DIP = 45（900 Hz 偏移，实际 100 + 45×20 = 1000 Hz）
- 想输出 **5 kHz 方波**：sw[7:5] = 010，DIP = 50（1000 Hz 偏移，实际 4000 + 50×20 = 5000 Hz）

---

## 五、幅度调节（有效值 0–2Vrms）

使用 **PB0（幅度+）** 和 **PB1（幅度−）** 调节输出信号幅度，以有效值（Vrms）标定。

| 按键 | 操作 | 步进 | 范围 |
|:---:|:---|:---:|:---:|
| **PB0** | 幅度增加 | +10（约 +78mVrms） | 0 ~ 250 |
| **PB1** | 幅度减小 | −10（约 −78mVrms） | 10 ~ 255 |

### 幅度说明

- 幅度值范围为 **0 ~ 255**（8 位数字量），默认值 **255**（对应满幅度 2Vrms）
- 经 DAC 双极性电路转换输出（DAC Vref = 3.3V 实测，满幅 ±3.3V）
  - 0x80 (128) = 0V（中心电平）
  - 幅度 255 对应 DAC 满幅输出（±2.83V 峰值，已校准）
- **不同波形的最大有效值（Vrms）：**
  | 波形 | 最大值（理论） | 说明 |
  |:---|:---:|:---|
  | 正弦波 | ≈ 2.00Vrms | **≥ 2Vrms**，满足设计指标 |
  | 方波 | ≈ 2.83Vrms | **≥ 2Vrms**，Vrms = Vpeak |
  | 三角波 | ≈ 1.64Vrms | Vrms = Vpeak/√3，受物理限制无法达到 2Vrms |
- 幅度超过 DAC 量程时自动饱和限幅，不会产生回绕失真
- 按住 PB0/PB1 可持续调节（带 10ms 按键消抖）

---

## 六、调制功能

当 sw[4:2] 设为 011/100/101 时，进入调制模式。

### 6.1 AM 调幅（sw[4:2] = 011）

- **原理**：用 256 点正弦调制波控制载波幅度
  ```
  输出 = 128 + (载波 − 128) × 调制系数 / 128
  调制系数 = 128 + 调制深度 × (调制波 − 128) / 256
  ```
- 屏幕显示：载波绿色波形 + 调制包络（暗绿色闪烁点）

### 6.2 FM 调频（sw[4:2] = 100）

- **原理**：用正弦调制波偏移 DDS 频率调谐字（FTW）
  ```
  FM_FTW = 基频FTW + 调制深度 × 调制波值 / 256
  ```
- 载波频率围绕基频上下偏移

### 6.3 SPWM 正弦脉宽调制（sw[4:2] = 101）

- **原理**：载波（正弦）与内部三角波比较
  ```
  载波 > 三角波 → 输出 0xFF（高电平）
  载波 < 三角波 → 输出 0x00（低电平）
  ```
- 输出为数字 PWM 信号（只有 0x00 和 0xFF 两个电平）

### 调制深度调节

| 按键 | 操作 | 步进 | 范围 |
|:---:|:---|:---:|:---:|
| **PB2** | 调制深度增加 | +16 | 0 ~ 240 |
| **PB3** | 调制深度减小 | −16 | 16 ~ 255 |

- 默认调制深度：**128** (0x80)
- 调制深度越大，调制效果越明显
  - AM 模式：深度越大 → 包络起伏越大
  - FM 模式：深度越大 → 频偏越大
  - SPWM 模式：深度影响三角波比较（通过 tri_cnt 间接影响）

## 七、VGA 屏幕显示说明（信号发生器模式）

进入信号发生器模式后，VGA 屏幕显示如下内容：

| 显示元素 | 颜色 | 说明 |
|:---|:---|:---|
| 网格线 | 暗灰色 (RGB: 2,2,2) | 每 80 像素列、60 像素行一条线 |
| 中心十字 | 亮灰色 (RGB: 4,4,4) | 屏幕中心 (320, 240) 处十字准线 |

## 九、快速上手指南

### 场景一：输出 1kHz 正弦波

1. 将 **sw[1:0]** 拨到 **00**（信号发生器模式）
2. 将 **sw[4:2]** 拨到 **000**（正弦波）
3. 将 **sw[7:5]** 拨到 **000**（100 Hz 基频档）
4. 将 **DIP[7:0]** 设为 **00101101**（45 × 20 = 900 Hz，总输出 = 1000 Hz）
5. 按 **PB0** 调节到所需幅度

### 场景二：输出 5kHz 方波

1. 将 **sw[1:0]** 拨到 **00**
2. 将 **sw[4:2]** 拨到 **001**（方波）
3. 将 **sw[7:5]** 拨到 **010**（4 kHz 基频档）
4. 将 **DIP[7:0]** 设为 **00110010**（50 × 20 = 1000 Hz，总输出 = 5000 Hz）

### 场景三：输出 AM 调幅波

1. 将 **sw[1:0]** 拨到 **00**
2. 将 **sw[4:2]** 拨到 **011**（AM）
3. 设置频率（同上）
4. 按 **PB2/PB3** 调节调制深度
5. 观察屏幕上绿色包络线的变化

## 十、注意事项

1. **频率档位切换**：切换 sw[7:5] 时，输出频率会立即跳变，建议先断开负载再切换
2. **调制模式**：进入 AM/FM/SPWM 模式（sw[4:2]=011/100/101）会自动开启调制，退出后自动关闭
3. **幅度饱和**：幅度最大值 250（+10 步进不超过 250），最小值 10（−10 步进不低于 10）。当幅度设置超过 DAC 量程时，输出自动饱和限幅，不会产生回绕失真
4. **按键消抖**：所有按键均经过 10ms 消抖处理，快速连按不会误触发
5. **DAC 输出**：输出为双极性信号（最大 ±3.3V），幅度以 Vrms 标定（0–2Vrms），请勿直接驱动低阻抗负载
6. **频率上限**：DDS 时钟为 25 MHz，理论最高输出频率受奈奎斯特限制（< 12.5 MHz），实际可用频率最高约 13 kHz（受限于 sw 档位 + DIP 组合）
